Foxhole Global HQ

Foxhole Global HQ is a multi-user tool that allows teams to coordinate logistical efforts in the game.
Features include:

  • Real time map updates
  • User management
  • Tracking of refinery/production facility levels
  • Tracking of stockpiles/storage depots
  • Custom outpost markers with stockpiles
  • Resource refining calculator + timer
  • Squad tab for organizing roles and vehicle crews
  • Item calculator
  • Logistics requests
  • Blueprint menu and timers
  • Tech tree
  • Notes for markers

Guide

1. Profile

After authorizing through Steam, an account for the website is created and submitted into the database. Profiles are used to manage access to the rooms since a lot of sensitive information can be stored there. On your profile page you can see the rooms you are part of(1) Leaving a room you are an admin of will also delete the room. You can also create a new room(2). On the top right are navigation buttons(3). The panel on the top of the page is present on every page.

2. Room interface

The room interface consists of 4 main parts:

  • The User list (1)
  • The map (2)
  • The management panel (3)
  • The tabs (4)
2.1 User list

The user list displays users that are part of this room. Black background means the user is offline, green blackground means the user is online. The frame around the user name indicates the role of the user:

  • Orange - admin
  • Gold - moderator
  • White - user

Admins can manage all users, delete the room and transfer ownership. Moderators can manage all users except the admin and other moderators. Users have access to the room and change data in it.

2.2 Map

The map consists of icons that represent various objects around the map. Every 3 seconds the server makes a request to the WarAPI, and, if there is new data, updates the map and sends the new data to all users. The icons on the map are divided by layers - you can turn on and off any layers if you want.

Hovering your mouse over an icon will display the tooltip. For most of the objects the tooltip displays notes and the time of last update. For refineries and factories the tooltip additionally displays the facility levels. This data cannot be automatically obtained from the game and is updated by users, thus the time of last update.

Right clicking on the map will bring up a context menu that allows you to put various icons on the map.

2.3 Region pan menu

The region pan menu allows you to instantly pan to any region by clicking one of the buttons.

2.4 Facility filters

Selecting a facility filter will hide all town icons except those that contain all of selected facilities. This means selecting both basic material and refined material filters will leave only towns on the map where both facilities are present. This filter also hides all fort and observation tower icons while active.

3. Management panel

3.1 Town panel

After clicking on a town, the town panel will appear and a selection marker will be shown at town's location. The gray circle around the town hall or fort indicates the defense power radius.
The town panel consists of:

  • Refinery panel
  • Production panel
  • Stockpile panel
  • Storage panel
  • Notes panel
  • Date of last update

Refinery and production panels contain information about the levels of facilities in the selected town. You can enter a number to edit the data or press the "Fill" button. Pressing the "Fill" button will mark the facility as complete and increase upgrade part requirement for further upgrades.
The stockpile and storage panels can contain information about items in stockpiles/storage depots. You can click on any item and enter a number to edit the data. The changes will be immediately sent to the server. The filter buttons allow users to filter buttons by categories. Pressing the "Empty" button will reduce the amount of all items to zero.

The "Notes" section is present on all panels and can contain miscellaneous information about the selected item. Pressing "Save notes" will send the changes to the server. The "Last update" section contains information which says when was the last time this object has been updated.

3.2 Fort/Outpost panel

Fort panel is similar to the town panel, except it doesn't have the refinery/production facility panels. Outposts (FOBs) have an X button that allows you to delete them. You can change fort's faction and level in the menu.

3.3 Logi request panel

Logi requests can be placed anywhere on the map via a context menu. On the modal you can select the items for the request and set their priorities. You can select the amount of requested items via an input. Selecting 0 amount will mean the request is indefinite and will show up as "some".

The panel itself says who is the creator of the request and has "Manufacturers" button(1) and "Edit request" button(2). Only the creator of the request can edit the request. Clicking on the "Manufacturers" button will show who is currently working on items for the request.

Clicking on the logi request items will transfer them to your production table. Everyone can see what items you are currently producing and coordinate their efforts. Clicking on items in production table will transfer them to the submit table. Alternatively, pressing "Open submit menu" button will transfer all items from production to submit table. Adding any items to the submit menu automatically collapses the logi request panel to give more space.

Pressing "Submit items" will move the items from the submit table to the request table and mark it with green color. If the amount of a supplied item satisfies the request with the top priority, it will be crossed out. If it does not, the request will display the amount of item required and total amount requested on hover.

3.4 Artillery

On the bottom right corner of the map is the artillery panel. Click on it to enable the artillery calculator. The artillery calculator allows you to calculate the required azimuth and distance based on the data from your binoculars. You can send the calculations to the rest of the room users by clicking the Share result button.

You can click on any of the artillery icons on the bottom left panel to spawn an according icon at the center of your screen. If you can't see the icon, don't worry! They only show up at medium and high zoom levels. You can drag the icon around and see its range. You can click at any point withing the artillery's radius to see the range and azimuth for that point. With enough precision, a squad can use a set of howitzers using this map without binoculars. Disclaimer: the development team does not guarantee firing accuracy based on the information from this map. Field artillery also has to take ballistic arc into account.

4. Squads management

The squad tab allows you to set up squads, roles and configure vehicle crews.

You can create new squad by clicking "Create squad". You can change the name of the squad if you are the leader, or leave it. Other people can either join your squad or select their own.
Below the squad panel you can select your role and see how many people with each role are currently online.
On the bottom is the army composition menu that allows you to configure the vehicle setup for an operation and assign crews. Click on any of the categories to select it, and then click on the arrow to add a new vehicle.

5. Refinery tab

Refinery tab allows you to calculate refining times and ratios for resources and set up timers in your refinery of choice.

6. Logi calculator

Selecting different item/s will calculate and display on top the amount of different materials required to produce them.

7. Tech tree

The tech tree tab allows you to record information about the status of the technlogy tree. Select any technology and use the input or the "Fill" button to change the status of a technology. The rectangles on the background display the progress of technologies. The completed ones are marked with green.

8. Statistics

The statistics tab allows you to see the current state of the war. The Win Condition panel shows you victory town ownership, their levels and how many of them have been upgraded to civic centers. The progress bar in State of the war indicates the ratio of owned town halls and forts.

9. Settings tab

When you give a user a link to your room, he will be redirected to the "Request Access" page. There he can ask to join your room.

When the request is submitted, a notification will appear on your user management tab and you can choose whether to grant or deny access to the user. If you want, you can click on his name and check his Steam page first. Approving the user will give him a rank of viewer, denying the user will ban him.

When the user is approved, his name will appear on the users tab. Additionally, you can change the rank of the user whenever you want. The data will be sent to the server and other users in real time.

If you choose to make a user an admin, a warning modal will appear and inform you that making a user an admin will transfer the ownership to him. You cannot make an anonymous user the admin.

If you agree, the selected user will be promoted to room admin and you will be demoted to moderator.

9.1 Room settings

In room settings you can change the room name, password and toggle it to secure or unsecure. Setting it to secure will kick all anonymous users. Setting it to unsecure will allow access to anyone with the link and the password.
You can also link the Foxhole Global Bot to a discord channel using the instructions on the management page. You can access these features only if you are a Steam-logged user.
